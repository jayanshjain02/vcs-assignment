# Assignment it is 

## Overview
This is a simple project created for the Version Control Systems assignment. It demonstrates Git basics, branching, conflict handling, documentation, automation, and GitHub workflows.

## Installation
1. Clone the repository: git clone git@github.com:jayanshjain02/vcs-assignment.git

2. Navigate into the project folder: cd assignment


## Features
- Simple project structure
- Clean Git workflow demonstration
- Branching + Merging + Conflict resolution
- GitHub Issues, Milestones, and Documentation
- GitHub Actions CI
